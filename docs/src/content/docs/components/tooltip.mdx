## Usage

<Tabs>
	<TabItem label="Builder">
```svelte
<script lang="ts">
	import { Tooltip } from "melt/builders";

	const tooltip = new Tooltip();
</script>

<button {...tooltip.trigger}>
	Hover me
</button>
{#if tooltip.open}
	<div {...tooltip.content}>
		<div {...tooltip.arrow}></div>
		<p>Content</p>
	</div>
{/if}
```
	</TabItem>
